#React Native banner的实现

##用到的组件以及方法

  • ScrollView，用于banner的展示
  • TimerMixin定时器

##Banner的实现

分析需求

  1. banner中滚动的图片，其实就是一个水平的scrollView，内容为一张一张的图片，然后通过定时器让他定时滚动

  2. banner最下方是一个View，

  • 其中左边部分是图片对应的文字解释,是一个Text
  • 右边是指示器，圆点个数与图片个数相同，两种颜色的圆点其实也是文字, &bull； 他是html中特殊的符号，显示出来就是一个圆点，我们通过设置字体颜色就可以显示出两种不同的指示器点了,是一个View里面包裹了一个Text和指示器View指示器中包裹着圆点的Text

###实现过程

  • 写在前面， 我们看看state和props中有哪些我们需要用到的自定义属性

  1. 获取网络数据，并且将数据保存在state中，

    getBannerData(){
    fetch('http://www.mywebsite.com/getBanner')
     .then((response)=>response.json())
     .then((responderJson)=> {
        var jsonData = responderJson.data;
        this.setState({
            bannerArray: jsonData,
            //让banner左下角默认显示第一个文字描述
            bannerText: jsonData[0].banner_desc
        })
    })
    .catch((error)=> {
        console.error(error);
    });
},

    这里接口替换成你们的获取banner图片和文字描述的接口

  1. 代码中添加ScrollView组件

  1. 为scrollview中添加图片

    建立一个数组，数组中添加图片，图片的uri为第二步网络请求中得到的数据，并将数据返回给scrollview

**到目前为止，banner中的图片已经被我们实现出来了，我们接着实现图片下方的View**

  1. 下方文字提示以及指示器的view

    onMomentumScrollEnd={(e)=>this.onAnimationEnd(e)}

       onAnimationEnd(e){
    //计算出scrollView水平滚动的距离
    var offsetX = e.nativeEvent.contentOffset.x;
    //计算出当前页数,同时更新当前页数currentPage
    var currentPage = Math.floor(offsetX / width)
    var bannerText = this.state.bannerArray[currentPage].banner_desc;
    this.setState({
        currentPage: Math.floor(offsetX / width),
        bannerText: bannerText
    })
},


renderBannerPoint(){
    var pointArray = [];
    for (var i = 0; i < this.state.bannerArray.length; i++) {
    //通过图片的位置计算出的当前页数来改变圆点的颜色，如果等于当前颜色就设置白色，证明是当前页数,反之
        var fontColor = (i == this.state.currentPage) ? {color: 'white'} : {color: 'rgba(220,220,220,0.8)'};
        pointArray.push(<Text key={i} style={[styles.bannerPointStyle, fontColor]}>&bull;</Text>)
    }
    return pointArray;
},

    var TimerMixin = require(‘react-timer-mixin’);
    mixins: [TimerMixin],

    startTimer(){

      //引入scrollView便于后续操作
var scrollView = this.refs.bannerScroll;

this.timer = this.setInterval(function () {

    //设置指示器圆点变换
    var timerPage = 0;
    if (this.state.currentPage + 1 >= this.state.bannerArray.length) {
        timerPage = 0
    } else {
        timerPage = this.state.currentPage + 1;
    }
    this.setState({
        currentPage: timerPage
    })

    //设置图片滚动变换,通过scrollResponderScrollTo可以滚动图片
    var offsetX = timerPage * width;
    scrollView.scrollResponderScrollTo({x: offsetX, y: 0, animated: true})
    //intervalTime为我在props中设置的时间间隔属性
}, this.props.intervalTime);

    },
