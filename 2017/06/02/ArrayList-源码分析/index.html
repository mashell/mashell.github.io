add() 方法

用于增加元素.他会将元素放在末尾

当增加数据的时候,如果容量不够,就会进行扩容,扩容成原来大小的1.5倍,然后把老的数组拷贝到新的数组里面.

get() set() 方法

两个方法比较简单,首先检查 index 坐标,然后都是数组操作.

remove() 方法

remove() 用于移除元素,你可以传入 index 或者传入元素值

remove()原理也不难,利用System.arraycopy(),将后面的元素向前移动,覆盖前面的元素达到删除的目的.
